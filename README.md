# Raspberry Pi 5 Projects

## Overview
A collection of projects built and running on a Raspberry Pi 5,
covering networking, security, and biomedical signal processing.

## Projects

### 1. Pi-hole (Network-Wide Ad Blocker)
- Blocks ads and tracking domains for every device on the network
- Configured as DHCP server for entire home network
- Uses StevenBlack blocklist + Cloudflare DNS
- Web dashboard for real-time network monitoring
- 📁 See: /pihole

### 2. WireGuard VPN
- Self-hosted VPN server running on Raspberry Pi 5
- Encrypted tunnel for secure remote access
- Integrated with Pi-hole for ad blocking on the go
- Protects traffic on public WiFi networks
- 📁 See: /wireguard

### 3. ECG Biomedical Signal Processing (In Progress)
- Real-time ECG signal acquisition using AD8232 sensor
- Digital filtering (low-pass + 60Hz notch filter)
- FFT frequency domain analysis
- Built for Electrical Engineering portfolio
- 📁 See: /ecg-signal-processing

## Hardware
- Raspberry Pi 5 (4GB RAM)
- Ethernet connection to home router
- Rogers Helix home network
